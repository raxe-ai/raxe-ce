# Jupyter Notebook Examples

Interactive RAXE tutorials in Jupyter notebooks.

## Contents

- `raxe_tutorial.ipynb` - Complete RAXE tutorial with visualizations

## Usage

```bash
pip install jupyter raxe pandas matplotlib
jupyter notebook raxe_tutorial.ipynb
```

## Features

- Interactive scanning examples
- Batch processing with pandas
- Visualization with matplotlib
- Decorator pattern demonstrations
