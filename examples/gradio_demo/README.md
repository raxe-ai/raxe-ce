# Gradio Demo with RAXE

Interactive web interface for RAXE threat detection demonstrations.

## Features

- Real-time text scanning
- Visual threat indicators
- Example prompts
- Detailed detection information

## Usage

```bash
pip install gradio raxe
python app.py
```

Opens at `http://localhost:7860`

## Demo Features

- Paste any text to scan
- See severity levels with color coding
- View detection details
- Try example prompts
- See scan metadata

## Share Demo

```bash
# Create public link
python app.py --share
```
