# Assets Directory

This directory contains static assets for RAXE Arena.

## Current Assets

- **No external assets required** - The game uses:
  - Unicode emoji for icons
  - CSS-based graphics
  - ASCII art for branding
  - Web-safe fonts

## Future Asset Plans

If you want to add custom assets:

### Icons
- Achievement badges (SVG recommended)
- Rank emblems
- UI icons

### Graphics
- Level completion animations
- Background patterns
- Loading animations

### Audio (Optional)
- Success sound effects
- Button click sounds
- Background music

## File Formats

Preferred formats for web performance:
- **Images:** WebP (fallback to PNG)
- **Icons:** SVG
- **Audio:** OGG (fallback to MP3)
